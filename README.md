# 🏴‍☠️ SuperPirate

**SuperPirate** is a 2D Java platformer game where players control a pirate navigating through levels, avoiding obstacles, and interacting with game entities.

The project demonstrates the core components of a custom-built Java game including **entity systems, level handling, input management, and UI rendering**.

---

## 🎮 Gameplay

In **SuperPirate**, the player controls a pirate character exploring levels filled with obstacles and interactive elements. The goal is to navigate through the environment while mastering movement and game mechanics.

---

## ✨ Features

* 🏴 Pirate-themed 2D platformer
* 🎮 Keyboard input system
* 🧱 Tile-based level structure
* 👾 Entity-based game objects
* 🎨 Custom UI system
* 📦 Organized resource management

---

## 🛠 Tech Stack

| Technology           | Purpose                    |
| -------------------- | -------------------------- |
| **Java**             | Core game logic            |
| **Java 2D / Swing**  | Rendering and UI           |
| **Custom Game Loop** | Game updates and rendering |
| **Resource System**  | Game assets and levels     |
